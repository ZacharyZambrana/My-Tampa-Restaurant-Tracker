Chatbot Embed Area

Your Botpress chatbot will be embedded here.

Instructions to embed your bot:

  1. Create and publish your Botpress bot
  2. Copy the embed code from Botpress
  3. Paste the code just before the closing </body> tag
  4. The chatbot will appear in this container

How to Set Up Your Botpress Bot

  1. Create Botpress Account: Sign up at https://botpress.com/ using your USF email
  2. Create a Table: Add your restaurant data with 10+ entries including Name, City, Cuisine, Rating, and Price
  3. Build Knowledge Base: Create a knowledge base from your restaurant table so the bot can search it
  4. Set Up Workflow:
    • Start Node → Autonomous Node
    • Add instructions like "You are a helpful assistant that answers questions about restaurants"
    • Enable Knowledge Base search
  5. Test Your Bot: Ask questions about restaurants to verify it works
  6. Publish & Embed: Publish your bot and copy the embed code here
