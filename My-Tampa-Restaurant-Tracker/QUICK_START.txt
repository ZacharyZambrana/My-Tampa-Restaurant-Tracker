🔧 BOTPRESS SETUP (For Chatbot Integration):

1. Go to https://botpress.com/
2. Sign up with USF email (@usf.edu)
3. Create table with restaurants data (10+ rows)
4. Build knowledge base from table
5. Create workflow: Start Node → Autonomous Node
6. Add instructions: "You are a helpful restaurant assistant..."
7. Enable "Search Knowledge"
8. Test your bot with restaurant questions
9. Publish and copy embed code
10. Paste code in mybot.html (before </body>)
